# DoubleApple

DoubleAppleは、原文と訳文の「手動翻訳」を強力に支援するテキストエディタアプリケーションです。左右に並んだエディタ間で翻訳の対応関係を視覚的に結びつけ、ホバーによるハイライト表示を行うことができます。

---

## 🚀 ビルド済み成果物（`out`）の使用方法

本リポジトリには、すでに静的HTMLとしてビルドされた成果物（[out/]ディレクトリ）が含まれているため、ダウンロードしてすぐ実行できます。

### ローカルPCでの実行
Node.js をインストールし、リポジトリのルートディレクトリで以下のコマンドを実行することで簡易Webサーバーが立ち上がり、ローカルで動作させることができます（※ダブルクリックで直接 `index.html` を開いても動作しません）。

```bash
npx serve out
```
コマンド実行後、ブラウザで `http://localhost:3000` (または表示されたポート番号) にアクセスしてください。

## 💡 アプリケーションの概要

手動での翻訳作業や対訳データの作成時に、どの単語や文章が互いに対応しているかを「タグ」として登録・ハイライト表示し、確認作業を効率化します。

### 主な機能
- **対訳対応タグの作成と連動ハイライト（ホバータグ）**
  - 片方のエディタ上のタグにマウスカーソルを合わせると、もう片方の対応するテキスト範囲が連動して緑色にハイライトされます。
- **ファイルのインポート・エクスポート**
  - `.txt` および `.pdf` ファイルからテキストを抽出し、エディタに直接読み込めます。
  - 作成した対応データやテキスト状態は、独自フォーマット（`.dapl` ファイル）として保存・復元できます。

### 操作用ショートカットコマンド（エディタ内）
- `Ctrl + Enter` : 選択範囲を「対応タグ」として紐付け（対応モード時）
- `Ctrl + Shift + Enter` : 既存のタグ範囲を拡張
- `Ctrl + -` : 選択範囲と重なるタグを部分解除
- `Ctrl + Shift + -` : 選択したタグを完全に削除
- `Tab` : 編集モードと対応モードの切り替え
